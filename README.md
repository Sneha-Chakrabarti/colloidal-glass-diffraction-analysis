# colloidal-glass-diffraction-analysis

Computational reproduction and extension of the colloidal glass diffraction analysis series by Amelia C. Y. Liu et al.:

- Liu et al., *Phys. Rev. Lett.* **116**, 205501 (2016)
- Liu et al., *PNAS* **114**, 10344 (2017)
- Liu et al., *Science Advances* **8**, eabn0681 (2022)
- Liu et al., *Acta Cryst. A* **82**, 4 (2026)

The toolkit implements angular correlation analysis, projected bond-orientational order (BOO) decomposition, centrosymmetry mapping, and continuous Burgers vector calculation for disordered materials, all from first principles in pure Python/NumPy.

---

## Physics background

Amorphous materials like glasses lack long-range order, so their structure cannot be described by conventional crystallography. Local structure (the arrangement of a particle's nearest neighbors) controls many macroscopic properties: glass-forming ability, mechanical stability, aging, and plasticity.

The central idea of this toolkit is that **limited-volume diffraction patterns** (obtained by scanning a small X-ray or electron beam over a thin glass specimen) carry angular correlations that encode the symmetry of local polyhedral clusters. These correlations can be extracted via the **angular autocorrelation function** and decomposed into contributions from archetypal polyhedra (FCC, BCC, HCP, ICO, SC) using nonnegative least squares.

This is the projected analog of the 3D Steinhardt–Nelson–Ronchetti bond-orientational order parameters, extended to transmission diffraction geometry.

## Milestones

### Milestone 1: Projected BOO fingerprints (PRL 2016)

Reproduces Fig. 3 of PRL 2016. For each of five archetypal clusters, computes the rotationally averaged Fourier coefficients of the angular autocorrelation function, which serve as the projected BOO symmetry fingerprint.

Key result: each cluster has a unique fingerprint. ICO has depressed, flat even-n symmetries. FCC/BCC/SC are dominated by n=2,4,6. HCP peaks at n=6.

Figures generated:
- `figures/fig3_prl2016.png`: fingerprint bar charts for all five clusters
- `figures/fig3_convergence.png`: convergence of FCC fingerprint with number of orientations
- `figures/fig3_overlay.png`: all clusters overlaid on one axes

### Milestone 2: NNLS decomposition (PNAS 2017)

Reproduces Fig. 4 and Fig. 5 of PNAS 2017. Simulates colloidal glass symmetry spectra for five glass types (no additives, salt, surfactant, sedimented, random) and decomposes each into FCC/BCC/ICO/RAN contributions using nonnegative least squares with confidence ranges.

Key result: the polyhedral population varies systematically with interparticle potential. Surfactant glasses are ICO-dominated; no-additive glasses show BCC+FCC competition; sedimented glasses are mostly random.

Figures generated:
- `figures/fig4_pnas2017_spectra.png`: symmetry spectra with NNLS fits
- `figures/fig4_pnas2017_decomp.png`: component weights with error bars
- `figures/fig4_pnas2017_normalized.png`: normalized stacked bar chart

## Core equations

**Angular autocorrelation** (Eq. 1, PRL 2016):

```
C(q, Δ) = [⟨I(q,φ)·I(q,φ+Δ)⟩_φ - ⟨I(q,φ)⟩²_φ] / ⟨I(q,φ)⟩²_φ
```

**Fourier coefficients of the intensity** (Eq. 8, PRL 2016):

```
I^n_q = f²(q) · Σ_{i,j} i^n · J_n(q·r_{ij}) · exp(-i·n·φ_{ij})
```

where r_{ij} and φ_{ij} are the projected pair separation and azimuthal angle.

**Normalized symmetry magnitudes** (Eq. 9, PRL 2016):

```
c^n_q / c^0_q = |I^n_q|² / |I^0_q|²
```

**NNLS decomposition** (Eq. 2, PNAS 2017):

```
C^n = [C^n_FCC, C^n_BCC, C^n_ICO, C^n_RAN]^T · [f_FCC, f_BCC, f_ICO, f_RAN],
      f_cluster ≥ 0
```

## Notes on implementation

The bottleneck is the orientational average (Eq. 9). The computation scales as O(K · N² · n_max) where K is the number of random rotations, N the cluster size (13 for most clusters), and n_max=12. For K=20,000 this takes ~35s per cluster on a single core. The key optimization is evaluating at a single representative q value (q_center of the first diffraction peak) rather than a full q-grid, which is valid because the fingerprint shape is q-independent across the first peak (verified in PRL 2016 Fig. 2).

For publication-quality figures, increase N_ORIENTATIONS to 160,000 in the reproduce scripts.

### Milestone 3: Centrosymmetry and anisotropy (Sci Adv 2022)

Reproduces Figs. 2, 3, and 4 of Sci Adv 2022. Simulates spatial maps of four structural parameters (stability, centrosymmetry ratio, eps_n, eps_s) during aging and deformation. Shear bands emerge as diagonal stripes of low centrosymmetry and high eps_n at 45 degrees to applied strain.

### Milestone 4: Burgers vector and quadrupolar strain (Acta Cryst 2026)

Generates a 2D amorphous packing, applies simple shear, fits local distortion tensors from nearest-neighbor configurations, and computes the continuous Burgers vector and quadrupolar field. Burgers magnitude anticorrelates with local centrosymmetry F_IS, confirming that lower-symmetry environments are softer.

Figures generated:
- `figures/acta_cryst_2026_glass.png`
- `figures/acta_cryst_2026_maps.png`
